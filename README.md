# 📝 Beyond - Modern Blog Platform

A modern, responsive blog application built with **Next.js 16**, **React 19**, **TypeScript**, and **Tailwind CSS**.

It features a clean UI, article filtering, search functionality, dark mode, related articles, reading time estimation, likes, comments, and responsive design.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/blog-app.git
```

Move into the project

```bash
cd blog-app
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```
